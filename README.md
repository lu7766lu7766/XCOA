# OA-BackEnd
辦公室系統 (前端)

## 前置安裝
```
yarn 
```

### 本地開發
```
yarn dev
```

### 輸出壓縮
```
yarn build
```

###[開發預設網址：http://localhost:3000/](http://localhost:3000/)

>##### /template/vue.html
>###### 需使用rewrite

### vue 測試component
